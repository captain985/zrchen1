# 2048 游戏

一个功能完整的2048小游戏，具有音效和流畅的动画效果。

## ✨ 功能特性

### 核心游戏机制
- 🎮 完整的2048游戏逻辑
- ⌨️ 方向键控制（↑↓←→）
- 🎯 相同数字方块自动合并
- 📊 实时分数显示和最高分记录
- 🏁 智能游戏结束检测

### 音效系统 🔊
- **移动音效** - 方块移动时的音响反馈
- **合并音效** - 方块合并时的音响反馈
- **胜利音效** - 达到2048时的胜利音乐
- **失败音效** - 游戏结束时的失败提示音
- 🔇 音量控制按钮（支持静音）
- 💾 音效设置保存到本地存储

### 动画效果 ✨
| 动画 | 说明 |
|------|------|
| `pop` | 新方块出现时的弹出动画 |
| `merge` | 方块合并时的缩放动画 |
| `pulse` | 2048方块的脉冲光晕动画 |
| `bounce` | 按钮悬停时的弹跳动画 |
| `shake` | 游戏震动反馈效果 |
| `fadeIn` | 游戏结束弹窗淡入效果 |

游戏使用浏览器的 `localStorage` 保存：
- 最高分数 (`best2048`)
- 音效设置 (`sound2048`)

数据在浏览器缓存清空前会一直保留。

## 🔊 音效系统详解

游戏使用 Web Audio API 合成音效，而不是预录音文件：

```javascript
// 音频上下文初始化
const audioContext = new AudioContext();

// 生成简单音调
const oscillator = audioContext.createOscillator();
oscillator.frequency.value = 400; // Hz
oscillator.type = 'sine';
```

音效频率设置：
- 移动: 400 Hz (低频)
- 合并: 600 Hz (中频)
- 失败: 200 Hz (很低)
- 胜利: 523-784 Hz (多频率序列)

## 🎯 游戏规则

1. **移动**: 按方向键移动方块
2. **合并**: 相同数值的方块会合并成2倍的数值
3. **新方块**: 每次移动后会在随机位置出现新方块
   - 90% 概率出现2
   - 10% 概率出现4
4. **胜利**: 达到2048方块
5. **失败**: 无法再进行任何移动时

## 🐛 已知限制

- 音效生成依赖 Web Audio API 支持（大多数现代浏览器都支持）
- 某些旧浏览器可能不支持所有CSS3特性
- 在非HTTPS下仍可正常工作（不同于某些其他API）
